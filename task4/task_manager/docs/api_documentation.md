# Task Manager API Documentation

## Endpoints

### GET /tasks
Returns a list of tasks.

### GET /tasks/:id
Returns a specific task by ID.

### POST /tasks
Adds a new task.

### PUT /tasks/:id
Updates an existing task by ID.

### DELETE /tasks/:id
Deletes a task by ID.
